# RFC 5545 Fixture Notice

This directory contains iCalendar fixtures based on examples from RFC 5545,
"Internet Calendaring and Scheduling Core Object Specification (iCalendar)".

The examples are used only by the `swift-ical` test suite to verify parser,
serializer, validation, and recurrence behavior against the published RFC.

RFC source:

- https://www.rfc-editor.org/rfc/rfc5545
